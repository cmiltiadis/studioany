Open tools & workflows for digital writing, notetaking, publishing, reference & library management

Public workshop at Social Space Kaymakkin

Constantinos Miltiadis
Workshop repository / Obsidian vault

Date
Dec 15, 2024 3:00 PM
Location
Social Space Kaymakkin
Nicosia, Cyprus

Workshop on open tools and workflows for writing, notetaking, publishing, reference and library management at Social Space Kaymakkin.

Workshop program

  1. Digital writing tools and workflows
    1. Why word processors suck.
    2. Writing in Markdown.
    3. Introduction to Obsidian for personal knowledge management.
    4. Brief introduction to Logseq, a FOSS alternative to Obsidian.
  2. Library and reference management.
    1. What is and what is not a library.
    2. Calibre for ebook management.
    3. Zotero for bibliographic reference management.
  3. Publishing
    1. Pandoc for document conversion.
    2. Hugo for static website building.
    3. Sandpoints Hugo framework

Tools & software

  • Markdown (open standard, lightweight markup plain-text language)
  • Software for Markdown Zettelkasten workflows:
    • Obsidian (Free software for writing and knowledge management)
    • Logseq (FOSS for bullet journaling, knowledge and task management, PDF annotation, etc.)
  • PanWriter Markdown text editor with Pandoc integration
  • Pandoc (FOSS, universal document converter)
  • Calibre (FOSS for ebook management)
  • Zotero (FOSS for bibliographic reference managment)
  • Hugo (FOSS for building static websites)
  • Sanpoints (FOSS Hugo theme; open infrastructure of experimental digital publishing)
