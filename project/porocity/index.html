Porocity

Collaborative procedural design project using shape grammars. ETH Zurich, 2013

Constantinos Miltiadis, Jinkyuk Choi, Adel Albloushi, Georgios Anagnostopoulos, Christian Braun, Kaihong Chen, Grigorios Grigoriadis, Abel Groenewolt, Periklis Kyriakidis, Valle Medina, Regina Mirzoyants, Yu-ting Sheng, Togo Takamura
Sep 30, 2013
Project trailer

Porocity is a collaborative procedural design project implementing shape grammars in Processing. Produced by the 2013 MAS CAAD class at the Chair for Digital Architectonics (née Chair for CAAD), during a programming course taught by Benjamin Dillenburger.
